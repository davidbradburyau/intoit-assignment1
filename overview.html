Project Overview

The idea I have is to develop an all-encompassing technology solution for a business’ particularly in creative industries. This solution should be repeatable and able to be sold as a package for a fee but also involve repeat support contracts for the supplier. The network/system should be robust and secure to minimise down time and to protect intellectual property of the business. It should be accessible anywhere in the world securely to enable employees not to be chained down to an office.
